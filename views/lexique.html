    LE LEXIQUE DES US

    Explorez le lexique spécifique des gangs à Los Angeles, où des termes comme "banger" pour membre de gang, "set" pour territoire et "strapped" pour armé sont souvent utilisés dans leur jargon distinctif. Plongez dans cet univers linguistique unique qui caractérise la culture des gangs dans la ville.  

    Squabble : C'est tout simplement une bagarre, si vous avez un problème à régler avec un autre membre ou approchant vous réglez ça en 1v1 dans un endroit tranquille. Ça commence avec une accolade rapide, voire la vidéo ci-dessous. (en l'occurrence ce sont deux individus de gangs rivaux qui se battent).

    Hood Hopper : Terme péjoratif pour designer quelqu'un qui a changé de gang.

    Flipside : C'est le fait de changer de gang, c'est quelque chose qui arrive souvent a new york comparé a Los Angeles.

    Put-on : Initiation pour intégrer un gang

    Put-off : bannissement d'un membre d'un gang du quartier

    Lacking : Se faire prendre en manque signifie se faire prendre tout seul, normalement sans arme.

    Whip : Voiture.

    Lick : Une source d'argent, le plus souvent en référence à un cambriolage ou un vol. 'On va se faire une pige'.

    Opps (Opposition) : Désignant les ennemis à son set.

    Slide : Bouger (se déplacer).

    Squabble : Bagarre en 1vs1. Cela peut concerné deux simples civils comme deux mecs du même set ou de sets ennemis.

    Blunt : Joint.

    Homie : Mec de son quartier.

    Goon : Terme désignant un soldat de son set, un mec h24 dans son set.

    Block/Hood/Set — Désignant un quartier.

    Pull Up : Débarquer dans un hood pour sauter un ennemi (avec arme ou non) ou attraper.

    Glock/Nine : Une arme à feu.

    Cops/Crash : Désignant la police.

    Lackin : Ne pas être armé, être en manque d'arme (Exemple : Renoi c'est chaud j'suis Lackin à VineWood).

    No Lackin : Être armé, manque pas d'arme sur sois (Exemple : No Lackin mon boy, dit leur de venir que je m'en occupe !!).

    CUZZ : Désignant un ami proche (Cousin).

    Blood : mot avec lequel s'appelle les set Bloods entre eux.

    Beef : Clash, embrouille ou désaccord avec quelqu'un ou une autre équipe.

    Fucked Up : Être défoncer sous drogue.

    Cracker : Désignant un blanc de manière péjoratif.

    Huslters : Débrouillard dans la rue.

    Tweakin : Être perdu, dire n'importe quoi.

    Junky : Client de drogues douces.

    Crackhead : Client de drogues dures.*

    Goofy Ass/Dumb : Terme désignant à insulter une personne stupide.

    Slag/Hoe : Pute/Salope.

    Bodies : Un ou des meurtres.

    Krab : Crip dissident. Slobs : Dissidence Bloods.

    Tramps : Dissertation de Gangster Crip.

    Maggots/Milkshakes/Mini Skirts : Dissidence Mafia Crip.

    Busters : Dissertation de Hustler Crip (c'est-à-dire Slauson Ave Hustler Crip).

    Huggies : Hustler Crip dissident. Snoovers/Snoochies/Hoochie > Coochies/

    Hot Cereal : Hoover Crip diss.

    Nappyheads/Napps : dissension de Neighborhood Crip.

    Twinkies : Rollin 20s NHB diss.

    Débardeurs : Tree Top Piru dissident.

    Rickets : Un autre terme péjoratif pour les Crips. "Regardez-les, c'est du rachitisme, du sang.

    Faketeens : Une dissertation dirigée vers les membres du gang de la 18ème rue.

    Fakestreets : Dissertation dirigée vers Grape Street.

    Barlems : Une dissertation dirigée vers R30s Harlem Crip.

    Gaymarks/Rice Crispies/Raymarks : Dissertation sur Raymond Avenue Crip.

    Cheese Toast(ie)s : Dissolution du Crip de la Côte Est.

    Flowers/Fagencia/Chorrenzia : Florencia 13 diss.

    Blues : Un autre nom pour les Crips, mais pas aussi péjoratif.

    cK : Crip Killer.

    bK : Blood Killer.

    pK : Piru Killer.
